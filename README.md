### 🔥 Why ClipSafe?
Stop uploading your private videos to random servers. In an era where data is the new oil, **privacy isn't a feature — it's a human right.**

ClipSafe is a powerful, professional-grade Video Editor and AI Media Platform that runs **entirely in your browser**. No accounts. No subscriptions. No cloud processing. **Your data never leaves your machine.**

---

## ⚡️ The "Zero-Upload" Magic
Most editors say they are "private." We mean it. We use **WebAssembly (Wasm)** and **FFmpeg.js** to process everything locally. 


---

## 🛠 Features that Blow Minds

- **💎 Pro-Editor**: A full-featured timeline editor with transitions, layers, and realtime preview.
- **🧠 AI Assistant**: Talk to your video. Ask questions, generate captions, and automate edits.
- **🚀 Professional Micro-Tools**:
  - `Trim` | `Compress` | `Convert` | `Reverse` | `Merge`
  - `Audio Visualizers` | `Subtitles` | `GIF Maker` | `Color Correction`
- **📂 No File Size Limit**: Since we process locally, your only limit is your disk space.
- **🌐 Works Offline**: Once loaded, ClipSafe doesn't need the internet. Edit in the middle of the ocean if you want.

### How to Run Locally

```bash
# Clone the repository
git clone https://github.com/BetelBuddy/clipsafe.git

# Install dependencies
npm install

# Start the dev server
npm run dev
```
